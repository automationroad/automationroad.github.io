自动修改时间

点击显/隐内容

“ ~/.vimrc 用户vim配置
“ 关于更新时间说明：退出时按 ‘wq’ 自动更新时间，不论是否有修改；按 ‘x’ 则文件有修改时更新， 按 ‘q’ 更新时间无变化
“ modify the last modified time of a file
function SetLastModifiedTime(lineno)
​ let modif_time = strftime(“%Y-%m-%d %H:%M:%S”)
​ if a:lineno == “-1”
​ let line = getline(9)
​ else
​ let line = getline(a:lineno)
​ endif

if line =~ ‘\sLast Modified:’
​ let line = strpart(line, 0, stridx(line, “:”)) . “: “ . modif_time
endif

if a:lineno == “-1”
​ call setline(9, line)
else
​ call append(a:lineno, line)
endif

endfunc
“ 修改shell脚本的更新时间
function SetLastModifiedTime_sh(lineno_sh)
​ let modif_time_sh = strftime(“%Y-%m-%d %H:%M:%S”)
​ if a:lineno_sh == “-2”
​ let line_sh = getline(8)
​ else
​ let line_sh = getline(a:lineno_sh)
​ endif

if line_sh =~ ‘\sLast Modified:’
​ let line_sh = strpart(line_sh, 0, stridx(line_sh, “:”)) . “: “ . modif_time_sh
endif

if a:lineno_sh == “-2”
​ call setline(8, line_sh)
else
​ call append(a:lineno_sh, line_sh)
endif

endfunc

“ map the SetLastModifiedTime command automatically
autocmd BufWrite .py call SetLastModifiedTime(-1)
autocmd BufWrite .sh call SetLastModifiedTime_sh(-2)
“ 加载文件头模板
run vim_template/vim_header

这里的最后一次修改时间，代码跟行数有关系，例如python的字段在第9行，shell的在第8行，修改模板的时候留意一下是否和自动修改时间对应的行数相同

安装插件

vim 安装插件的两种方法

第一种是自行下载插件，配置 .vimrc
第二种是使用自动管理vim插件的 vundle ，本文使用 vundle

安装插件管理器 vundle

使用vundle安装/卸载/更新插件

安装

到这一步已经可以使用 vundle 管理插件了，vundle管理vim有两种安装插件的方法，一个是命令行模式安装，一个是进入vim模式安装
一：命令行下键入：vim +PluginInstall +qall
二：在vim的normal模式中键入命令：PluginInstall

新增插件

如果要安装新插件，在vimrc中加入bundle，然后执行
:BundleInstall

卸载

去掉某些自己用不到的插件: 编辑.vimrc.bundles，注释掉插件对应Bundle行即可(加一个双引号),保存退出即可
“Bundle ‘fholgado/minibufexpl.vim’
如果想从物理上清除（删除插件文件），注释保存后再次进入vim
命令行模式，执行:
:BundleClean

更新

:BundleUpdate　（更新之后，可能需要重新编译一下YCM）

遇到过的一些坑

1、自动更新文件修改时间，自定义模板的一定要核对，最后一次修改时间所在的行，是否和 .vimrc 文件中函数修改的对应
2、插件安装慢，使用vim安装插件，下载过程非常缓慢，甚至可能会因为网络不好部分插件下载失败，相比git clone到本地慢，所以一键部署采用git直接克隆到本地

总结

1、vim插件丰富，插件功能的详细介绍需要到GitHub上了解，本文只是介绍安装为主，对于功能的使用还在摸索中，在编写的过程中也发现了一些新的好用的插件，各位道友有不错的插件欢迎QQ交流
2、到此，一个vimIDE就打造好了，我将上面操作涉及到的文件都上传到了GitHub，并做好了一个自动部署脚本，Ubuntu系统也适用，需要的朋友可以自行下载执行安装，再次强调，整个配置是针对单个系统用户的，不是全局配置，此项目是目前的使用的配置，以后有需求会更新，觉得有用的朋友给个 star
